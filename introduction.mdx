## What is Natro Macro?

Natro Macro is a powerful, open-source automation tool for Bee Swarm Simulator written in AutoHotkey. It automates gathering, quests, planters, bosses, and much more to help you progress in the game efficiently.

<Warning>
  **Download only from official sources:**
  - [GitHub Repository](https://github.com/NatroTeam/NatroMacro)
  - [Discord Server](https://discord.gg/natromacro)
  
  We do **not** have an official website. Beware of malicious copies from other sources.
</Warning>

## Key features

<CardGroup cols={2}>
  <Card title="Automated gathering" icon="leaf">
    Gather from up to 3 fields simultaneously with customizable patterns (Squares, Lines, Snake, and more). Configure field rotation, pattern size, gathering duration, and pack requirements.

  <Card title="Quest automation" icon="list-check">
    Complete quests from Polar Bear, Honey Bee, Black Bear, Brown Bear, Bucko Bee, and Riley Bee automatically. Configurable quest interruption and progress tracking.

  <Card title="Boss & bug runs" icon="bug">
    Fight Tunnel Bear, King Beetle, Stump Snail, Coconut Crab, Commando Chick, and Vicious Bee. Configurable baby detection and amulet modes for optimal loot.

  <Card title="Collector automation" icon="coins">

## Pattern system

Natro Macro includes built-in gathering patterns:

- **Squares** - Efficient square-based coverage
- **Lines** - Simple back-and-forth pattern
- **Snake** - Continuous serpentine movement
- **XSnake** - Advanced snake variant
- **Diamonds** - Diamond-shaped pattern
- **Fork** - Forked gathering pattern
- **Slimline** - Narrow coverage pattern
- **Stationary** - Minimal movement pattern

Custom patterns can be imported from the [community Discord](https://discord.gg/natromacro) or the [Paths-Patterns repository](https://github.com/NatroTeam/Paths-Patterns).

<Note>
  Pattern files can execute AutoHotkey code. Only download patterns from trusted sources like the official Discord #patterns channel or verified GitHub repositories.
</Note>

## System requirements

- **Operating System:** Windows (64-bit recommended)
- **Display Scale:** 100% (96 DPI) - required for image recognition
- **AutoHotkey:** Included (32-bit and 64-bit versions)
- **Roblox:** Latest version with specific graphics settings
- **Administrator Privileges:** May be required depending on installation location
