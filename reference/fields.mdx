## Overview

Bee Swarm Simulator features 17 unique fields where players can gather pollen. Each field has different characteristics, boosters, and recommended strategies.

## Field Boosters

Field boosters enhance gathering efficiency in specific fields. The macro tracks booster types and their stack counts for optimal farming.

### Booster Types

- **Blue Booster** - Enhances blue pollen collection
- **Red Booster** - Enhances red pollen collection  
- **Mountain Booster** - Enhances white/mountain pollen collection
- **None** - No booster available

## Field Booster Configuration

The macro uses the following FieldBooster map for automatic field selection:

```ahk
FieldBooster := Map(
    "pine tree", {booster:"blue", stacks:1},
    "bamboo", {booster:"blue", stacks:1},
    "blue flower", {booster:"blue", stacks:3},
    "stump", {booster:"blue", stacks:1},
    "rose", {booster:"red", stacks:1},
    "strawberry", {booster:"red", stacks:1},
    "mushroom", {booster:"red", stacks:3},
    "pepper", {booster:"red", stacks:1},
    "sunflower", {booster:"mountain", stacks:3},
    "dandelion", {booster:"mountain", stacks:3},
    "spider", {booster:"mountain", stacks:2},
    "clover", {booster:"mountain", stacks:2},
    "pineapple", {booster:"mountain", stacks:2},
    "pumpkin", {booster:"mountain", stacks:1},
    "cactus", {booster:"mountain", stacks:1},
    "mountain top", {booster:"none", stacks:0},
    "coconut", {booster:"none", stacks:0}
)
```

## Path Files

Each field has associated path files in the `paths` directory:

- **gtf-[field].ahk** - Go to field paths
- **gtp-[field].ahk** - Go to planter paths (for planter placement)
- **wf-[field].ahk** - Walk field paths (gathering patterns)

## Usage Tips

<AccordionGroup>
  <Accordion title="Maximizing Booster Stacks">
    Focus on fields with 3 stacks (Blue Flower, Mushroom, Sunflower, Dandelion) when you have the corresponding field booster active.
  </Accordion>

  <Accordion title="Quest Farming">
    Many quests require gathering from specific fields. The macro automatically navigates to quest-specified fields when quest gathering is enabled.
  </Accordion>

  <Accordion title="Field Selection Strategy">
    - **Early game**: Focus on Sunflower, Dandelion, and Blue Flower for high booster stacks
    - **Mid game**: Expand to Mushroom, Clover, and Bamboo fields
    - **Late game**: Include Pepper, Pine Tree, Spider, and Cactus fields
    - **End game**: Mountain Top and Coconut for specific resources
  </Accordion>

  <Accordion title="Movement Methods">
    Fields can be accessed via:
    - **Walk** - Standard pathfinding from hive/ramp
    - **Cannon** - Faster access using the cannon (requires proper timing)
    
    The macro automatically selects the appropriate method based on your `MoveMethod` configuration.
  </Accordion>
</AccordionGroup>
